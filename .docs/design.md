# タスクツリー アプリ設計

## アプリ概要

- タスクをツリー構造で管理するアプリ。
- 全体の工数（予想時間）を親タスクで定義し、それを子タスクに分割していく。

## データ設計

- **Task**
  - `id`: string (UUID or timestamp)
  - `title`: string (タイトル)
  - `estimatedMinutes`: number (予想時間：分)
  - `isCompleted`: boolean (完了フラグ)
  - `children`: Task[] (子タスクの配列)

### 工数制約

- `親の estimatedMinutes` >= `Σ (すべての子の estimatedMinutes)`
- 親タスクの時間を超えて子タスクを割り振ることはできない。

## 機能

- **ツリー表示**: ネストされたタスクの一覧。
- **WBS**: ガントチャート的な進捗管理（フェーズ2）。
- **バーンダウンチャート**: 残り工数の可視化（フェーズ3）。
