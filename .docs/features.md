# タスクツリー アプリ

タスクをツリー構造で管理し、工数（分）を階層的に割り振るアプリです。

## 実装済み機能

- **ツリー型タスク管理**:
  - 親タスクから子タスクを無限に作成可能。
  - 各タスクにタイトルと予想工数（分）を設定。
- **工数制約の自動バリデーション**:
  - 「子タスクの合計工数 ≦ 親タスクの工数」というルールを厳格に適用。
  - 親の工数を減らそうとした際、子の合計を下回る場合は自動でクランプされます。
  - 子の工数を増やそうとした際、親の空き時間を超える場合は自動で制限されます。
- **進捗管理 (WBS)的な表示 & 表示制御**:
  - 子タスクがないタスクにチェックボックスを搭載。
  - **子要素の折り畳み**: 子タスクを持つ親タスクに「畳む/開く」ボタン（▼/▶）を追加し、その配下の子要素をまとめて表示・非表示可能。
  - 子を持つタスクの進捗率は、そのサブツリー全体の完了済み工数から算出されます。
  - タスクごとにプログレスバーを表示し、視覚的に進捗を確認可能。
- **全体サマリー**:
  - ルートタスク全体の進捗（完了工数 / 総工数）を上部に表示。

## データ構造

```typescript
export interface Task {
  id: string;
  title: string;
  estimatedMinutes: number;
  isCompleted: boolean;
  childIds: string[];
  parentId: string | null;
}
```

## 今後の拡張性

- **バーンダウンチャート**: 完了した時間（タイムスタンプ）を記録することで、時間軸に沿ったバーンダウンチャートの描画が可能になります。
- **WBS詳細表示**: 開始日・終了日の概念を追加することで、ガントチャート形式の表示が可能になります。
